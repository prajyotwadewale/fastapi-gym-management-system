# 🏋️ Gym Management System (FastAPI)

## 📌 Project Overview

This project is a backend application built using FastAPI as part of the Innomatics Research Labs Internship.

It simulates a real-world Gym Management System where users can manage plans, memberships, and class bookings.

---

## 🚀 Features

* RESTful APIs using FastAPI
* Pydantic data validation
* CRUD operations (Create, Read, Update, Delete)
* Membership enrollment workflow
* Class booking system
* Search functionality
* Sorting of data
* Pagination
* Combined filtering and browsing APIs
* API testing using Swagger UI

---

## 🛠 Technologies Used

* Python
* FastAPI
* Uvicorn
* Pydantic

## ▶️ How to Run the Project

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Run the server:

```
uvicorn main:app --reload
```

3. Open Swagger UI:

```
http://127.0.0.1:8000/docs
```
